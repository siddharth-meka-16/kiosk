# Design System — Immersive Accessibility UI

## Style
- futuristic SaaS interface
- glassmorphism design
- dark gradient background
- premium startup feel

## Accessibility Focus
- large interactive elements
- high contrast
- readable fonts
- simple layout

## Visual Elements
- gradient orbs
- blurred glass panels
- soft glow borders
- depth and layering

## Animations
- smooth transitions
- staggered component entry
- hover scaling
- micro-interactions

Use framer-motion style animations.

## Layout Principles
- minimal clutter
- strong hierarchy
- guided flow
- fast interaction

## UX Goals
User should feel:
- supported
- confident
- not overwhelmed
