 # User Flow — Demo Optimized

## Entry
User lands on immersive landing page
→ clicks "Start Demo"

## Mode Selection
User selects:
- visually impaired
- hearing impaired
- beginner

## Menu
→ user sees simplified menu
→ selects item

## Cart
→ reviews order
→ proceeds to payment

## Payment
→ AI guides step-by-step
→ user selects payment method (simulated)

## Success
→ order confirmed
→ clear feedback displayed

Examples:

- beginner → simple instructions
- hearing impaired → clear text cues
