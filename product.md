# Product Overview — Kiosk Vision AI

## Mission
Build an AI-powered accessibility layer that transforms any self-service kiosk into an inclusive, guided, and frictionless experience.

## Positioning
This is NOT a kiosk system.
This is an intelligent accessibility layer that enhances existing kiosk interfaces.

## Target Users
- visually impaired users
- hearing impaired users
- elderly users
- first-time users unfamiliar with digital systems

## Core Problem
Current kiosks:
- are visually complex
- lack guidance
- exclude specially-abled users
- create confusion during ordering and payment

## Solution
Kiosk Vision AI:
- adapts UI based on user needs
- provides real-time AI guidance
- simplifies ordering and navigation
- assists users confidently through the entire flow

## Primary Outcome
User can:
→ complete an order independently within 1–2 minutes

## Demo Philosophy
- smooth
- intuitive
- emotionally impactful
- realistic but simplified

Focus on usability, not technical complexity.
